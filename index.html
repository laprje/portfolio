My First Project

AutoValue

A user can login or register a new account. To register, an $8 payment is required to access the site, for which I used Stripe payment processing. Users can search for any make, model, and year of car, and will recieve a ChartJS line-chart of the expected depreciation value of the selected car for the next 5 years. The pricing data is not static; the data is from a 3rd party API and updates based on the actual market values of the car. Users can compare up to 6 cars on one graph, and can save one search to their profile.

ReactJS | NodeJS | Express | Massive | Heroku | PostgreSQL | ChartJS | Stripe
